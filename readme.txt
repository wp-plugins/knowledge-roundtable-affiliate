Earn money by referring students to The Knowledge Roundtable. Easily show tutors from The Knowledge Roundtable's tutor directory on your site!

== Description ==

The Knowledge Roundtable affiliate program allows affiliates to earn money by introducing students or their parents to our tutoring service. This plugin allows you to display our tutor directory on your site. Earn commissions when customers follow links from the directory and pay for lessons on The Knowledge Roundtable. You must be a member of the The Knowledge Roundtable affiliate program to use this plugin. <a href="https://www.theknowledgeroundtable.com/affiliate-program/">Apply to The Knowledge Roundtable's affiliate program today.</a>

= Key features =

* Show tutors from The Knowledge Roundtable on any post or page with a shortcode.
* Embed a search form anywhere on your site with a shortcode or a widget.
* Search results auto-detect user location.
* Online tutors are shown to users when no local tutors are available.
* Free setup and integration support.

== Installation ==

You can install the plugin automatically from within your WordPress interface. Simply log in to WordPress, select Plugins from the menu and then Add New. Search for "Knowledge Roundtable Affiliate" and click the install link to automatically install the plugin.

For older versions of WordPress or manual installation: 1. Download and unzip the plugin zip file. 2. Upload the entire folder to the /wp-content/plugins/ directory of your WordPress installation.

Once you have the plugin installed, activate it through the WordPress ‘Plugins’ menu. Update your settings by navigating to Settings>KnowRo Affiliate. Enter your Tracking ID (TID) provided after registering for the program.
